Roadmap de Desenvolvimento — 3 Fases

Fase 1 — MVP (4–6 semanas)

  • ▲ Next.js + Vercel (substituir HTML estático)
  • 🗄️ Supabase: users + instagram_accounts + dashboard_data
  • 🔐 Auth: cadastro + login (email/Google)
  • 📱 OAuth Meta: conectar conta Instagram
  • 📊 Dashboard por conta (dados reais)
  • ⏰ Cron semanal de coleta automática

Fase 2 — Monetização (2–3 semanas)

  • 💳 Stripe: planos Free / Pro / Agency
  • 🔒 Limites por plano (nº de contas, features)
  • 🧠 IA Engine: análise + calendário automático
  • 📄 Exportação PDF + relatório IA por conta
  • 📧 Email semanal com resumo de performance
  • 👤 Painel admin para gerenciar clientes

Fase 3 — Escala (contínuo)

  • 🔔 Alertas: queda de alcance, melhor horário
  • 📊 Comparativo entre contas (Agency)
  • 🤖 Sugestões de conteúdo geradas por IA
  • 📱 App mobile (React Native)
  • 🔗 Integrações: Notion, Slack, WhatsApp
  • 🌍 Multi-idioma · White-label para agências
