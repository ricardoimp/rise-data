# Rise Data

Dashboard interativo de performance do Instagram de Ricardo Impellizieri.

## Funcionalidades

- 📊 Performance Geral (últimos 90 dias)
- 📅 Melhores Dias & Horários para postagem
- 🎬 Análise por Formato (Reels, Carrossel, Post Estático)
- 📝 Top Posts por Alcance e Engajamento
- 👥 Análise de Audiência
- 🎯 Funil de Conversão
- 💡 Insights Estratégicos Prioritários

## Tecnologias

- HTML5 + CSS3 (dark theme)
- Chart.js 4.4
- Deploy: Vercel
